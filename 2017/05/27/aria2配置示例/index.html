首先，aria2是个下载器。你要说为什么不用迅雷呢，这我也很难回答你

你总得把它安装上吧

aria2的一大特点是几乎所有系统都可以使用,通过以下命令随意安装

apt install aria2
or
yum install aria2
or
pacman -S aria2
or
brew install aria2
or
choco install aria2
